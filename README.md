This repo show the state of the expr shorthand work in progress, under Tcl-9.03.
It contains :
* the summaries of the changes made to Tcl-9.03 (file change-9.03.diff)
* A tclsh compiled with cygwin under windows (with its cygtcl9.0.dll)
* The full c files which were modified 
* An exemple (affine_transform.tcl) program that the modifications allow to write.
